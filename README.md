# Student Wellness Tracker

This application helps students track their academic load, daily time balance, and health habits to provide personalized wellness insights.

## Run Locally

**Prerequisites:** Node.js

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Run the app:**
    ```bash
    npm run dev
    ```
